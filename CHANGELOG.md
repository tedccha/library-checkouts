# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-06-29

### Added
- Enable holds fetching from library catalog with shared browser session, avoiding repeated Cloudflare challenges
- Display "On Hold" section on dashboard showing hold status and pickup dates

## [0.1.0] - 2026-06-27

### Added
- Initial release with Google OAuth authentication
- Fetch and display checked-out books with due dates and renewal counts
- Separate overdue and current books sections
- Book statistics dashboard
